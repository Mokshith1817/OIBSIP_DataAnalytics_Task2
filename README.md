## 📋 Project Overview

### Objective
Perform Customer Segmentation Analysis for 
an e-commerce company by analyzing customer 
behavior and purchase patterns using 
K-Means Clustering Algorithm.

### Dataset
Marketing Analytics: Customer Segmentation
(ifood_df.csv from Kaggle)

---

## 🔑 Steps Performed
1. Data Loading & Cleaning
2. Descriptive Statistics
3. Feature Selection
4. Data Scaling (StandardScaler)
5. Elbow Method (Finding Best K)
6. K-Means Clustering
7. PCA Visualization
8. Cluster Analysis
9. Recommendations

---

## 💡 Key Insights
- Income is strongest factor in segmentation
- 4 distinct customer segments identified
- High income customers respond better
  to marketing campaigns
- Family customers need different targeting
  than single customers
